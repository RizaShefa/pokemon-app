# 🧢 Pokémon Team Builder

A modern, fully responsive web application built with **Next.js 14**, **TypeScript**, **React**, and **Tailwind CSS** that brings the world of Pokémon to your fingertips. Discover, explore, and build your ultimate Pokémon team with an intuitive and lightning-fast user experience.

## 🌟 Key Features

### 🔍 **Smart Pokémon Discovery**
- **Advanced Search** – Find Pokémon instantly by name, ID, or type
- **Interactive Grid Layout** – Browse through beautifully designed Pokémon cards


### 📊 **Comprehensive Pokémon Details**
- **Detailed Stats Dashboard** – View HP, Attack, Defense, Speed, and more
- **Ability Descriptions** – Learn about each Pokémon's unique abilities

### 👥 **Personal Team Management**
- **Drag & Drop Team Builder** – Easily add/remove Pokémon from your team

### 🎨 **Premium User Experience**
- **Responsive Design** – Seamless experience across desktop, tablet, and mobile

### ⚡ **Performance & Reliability**
- **Optimized API Calls** – Efficient data fetching with caching strategies
- **Error Boundaries** – Graceful error handling with retry mechanisms


## 🛠️ Tech Stack

- **Framework**: Next.js 14 
- **Language**: TypeScript for type safety
- **Styling**: Tailwind CSS with custom components
- **State Management**: Context for global state
- **API**: PokéAPI integration with custom hooks
- **Deployment**: (https://pokemon-app-seven-orpin.vercel.app/)


 Getting Started
1️⃣ Clone the Repository
Using Create Next App:
```
https://github.com/RizaShefa/pokemon-app.git
npx create-next-app@latest -e https://github.com/your-username/pokemon.git pokemon-team
```
Or clone directly:
```
git clone https://github.com/your-username/pokemon.git

cd pokemon
```
2️⃣ Install Dependencies

Using npm:
```
npm install
```
Or pnpm:
```
pnpm install
```


Or yarn:
```
yarn install
```

3️⃣ Run the Development Server
```
npm run dev
```
```
pnpm dev
```
Or:

```
yarn dev
```
Open your browser and visit:

```
http://localhost:3000
```
